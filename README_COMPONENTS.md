# Component-Based Web Builder 🎨

**A revolutionary approach to web design** - Pick beautiful pre-designed components, customize them in seconds, and build stunning websites without spending hours adjusting values!

## 🚀 Why Component-Based?

Instead of manually configuring every property, you can:
- ✅ **Pick** from 30+ professionally designed components
- ✅ **Customize** with simple sliders and dropdowns
- ✅ **Build** complete websites in minutes, not hours
- ✅ **Export** production-ready HTML/CSS

### 🎨 Simple Customization

Each component can be customized with:

1. **📝 Content**
   - Text, titles, placeholders
   - Simple text input

2. **📏 Size**
   - Small, Medium, Large, Extra Large
   - One-click sizing

3. **🎨 Color Scheme**
   - 7 pre-defined schemes (Purple, Blue, Green, Red, Orange, Pink, Dark)
   - Custom color picker
   - Instant color changes

4. **💫 Shadow**
   - Enable/disable toggle
   - Intensity slider (0-100)

5. **🔲 Corners**
   - Border radius slider (0-50px)
   - Real-time preview

## 🎯 How to Use

### Quick Start (3 Steps!)

1. **SELECT** a component
   - Browse categories (Buttons, Cards, Forms, etc.)
   - Click any component preview

2. **CUSTOMIZE** it
   - Enter your text
   - Choose size and color
   - Adjust shadow and corners

3. **ADD** to page
   - Click "➕ Add to Page"
   - See it appear in preview!

### Workflow

```
Pick Component → Customize → Add → Repeat → Export
```

## 🎓 Tips & Tricks

1. **Mix and Match**: Combine different component styles
2. **Color Consistency**: Use same color scheme across components
3. **Size Hierarchy**: Use different sizes for visual hierarchy
4. **Shadow Depth**: More shadow = more emphasis
5. **Corner Roundness**: Consistent corners = cohesive design
